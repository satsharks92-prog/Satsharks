## Question 1 🟢
**Correct Answer: A) 5x² + 5x + 5**

**Step-by-step:**
(x² + 5x − 3) + 4(x² + 2)
= x² + 5x − 3 + 4x² + 8
= 5x² + 5x + 5.

**Why other options are wrong:**
- B) 5x² + 5x − 1 — computed −3 + 2 = −1 instead of −3 + 8 = 5 (forgot to multiply 2 by 4).
- C) Only 1x² — forgot to distribute 4 to x².
- D) 4x² — forgot the original x².

🪤 TRAP ALERT: B has −1 as the constant, which could come from computing −3 + 2 = −1 (forgetting to multiply 2 by 4).

💡 KEY INSIGHT: Distribute the 4 to every term inside the parentheses: 4(x² + 2) = 4x² + 8, not x² + 8 or 4x² + 2.

⚡ FAST METHOD: Distribute, then combine like terms by degree.

📱✍️ VERDICT: Hand-solve — basic distribution and combining.

## Question 3 🟡
**Correct Answer: B) 6x² − 7x − 20**

**Step-by-step:**
(3x + 4)(2x − 5) using FOIL:
F: 3x · 2x = 6x²
O: 3x · (−5) = −15x
I: 4 · 2x = 8x
L: 4 · (−5) = −20
Sum: 6x² − 15x + 8x − 20 = 6x² − 7x − 20.

**Why other options are wrong:**
- A) Missing the middle term (−7x) — only multiplied first and last.
- C) Used +15x + 8x = +23x — sign error on Outer term.
- D) Leading coefficient 5x² — computed 3 · 2 = 5 instead of 6.

🪤 TRAP ALERT: A is what you get if you skip FOIL and just do (3x)(2x) + (4)(−5). Always include the Outer and Inner terms.

💡 KEY INSIGHT: FOIL produces four terms. The middle term is the sum of Outer + Inner.

⚡ FAST METHOD: F = 6x², O + I = −15x + 8x = −7x, L = −20.

📱✍️ VERDICT: Hand-solve — standard FOIL, under 30 seconds.

## Question 19 🟡
**Correct Answer: C) 4ab**

**Step-by-step:**
(a + b)² = a² + 2ab + b².
(a − b)² = a² − 2ab + b².
Difference: (a² + 2ab + b²) − (a² − 2ab + b²) = 4ab.

**Why other options are wrong:**
- A) 0 — forgot the cross terms.
- B) 2b² — dropped the a² terms and miscounted.
- D) 2a² + 2b² — added instead of subtracted, or used wrong expansion.

🪤 TRAP ALERT: A assumes (a+b)² − (a−b)² = 0 because they "look similar." But the cross terms have different signs and don't cancel.

💡 KEY INSIGHT: This is a classic algebraic identity. Expand both squares and subtract carefully.

⚡ FAST METHOD: Alternatively, use difference of squares: A² − B² = (A+B)(A−B) where A = (a+b), B = (a−b). Then (a+b+a−b)(a+b−a+b) = (2a)(2b) = 4ab.

📱✍️ VERDICT: Hand-solve — algebraic identity, either method works in 15 seconds.

💡 KEY INSIGHT: Perfect square: (px + q)² = p²x² + 2pqx + q². The middle coefficient is 2pq.

⚡ FAST METHOD: √9 = 3, √16 = 4. k = 2 × 3 × 4 = 24.

📱✍️ VERDICT: Hand-solve — perfect square pattern.

## Question 32 🟡
**Correct Answer: B) x³ + 3x²y + 3xy² + y³**

**Step-by-step:**
(x + y)³ = (x + y)(x + y)².
(x + y)² = x² + 2xy + y².
(x + y)(x² + 2xy + y²) = x³ + 2x²y + xy² + x²y + 2xy² + y³ = x³ + 3x²y + 3xy² + y³.

**Why other options are wrong:**

## Question 33 🔴 — SPR
**Correct Answer: 3**

**Step-by-step:**
6x² − x − 2 ÷ (3x − 2):
Using factoring: need (3x − 2)(ax + b) = 6x² − x − 2.
Leading term: 3x · ax = 3ax² → 3a = 6 → a = 2.
Constant term: (−2)(b) = −2 → b = 1.
Check middle: 3x(1) + (−2)(2x) = 3x − 4x = −x. ✓
Other factor: (2x + 1). So a = 2, b = 1. a + b = 3.

🪤 TRAP ALERT: Students might try polynomial long division, which works but is slower. Matching coefficients is more efficient here.

💡 KEY INSIGHT: If one factor is known, the other can be found by matching the leading coefficient and constant.

⚡ FAST METHOD: Leading: 6/3 = 2 → a = 2. Constant: −2/(−2) = 1 → b = 1. a + b = 3.

📱✍️ VERDICT: Hand-solve — coefficient matching.

## Question 35 🔴
**Correct Answer: D) 21**

3/(x−2) + 2/(x+4).
