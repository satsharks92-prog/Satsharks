Answer: C) 0 only
6|x − 5| = k gives |x − 5| = k/6. An absolute value equation has exactly one solution when k/6 = 0 (the only
solution is x = 5). If k/6 > 0, there are two solutions. If k/6 < 0, there are no solutions.
Question 18 (Quadratic Equations — Discriminant)
Answer: 18
For −5x² + bx − 18 = 0 to have no real solution: b² − 4(−5)(−18) < 0 → b² − 360 < 0 → b² < 360 → b < √360 ≈
18.97. The largest positive integer is 18.
Question 19 (Exponential Functions — y-intercept)
Answer: D) Neither I nor II
f(0) = a(1 + 2.5b). For g(x) = a(2.5x + k): g(0) = a(1 + k). The y-intercept value a(1 + k) is not displayed as a

not displayed as a single visible constant. Neither form displays the y-coordinate as a constant or coefficient.
Question 20 (Linear Equations — Word Problems)
Answer: D) y = 95x + 85
Cost = $180 for the first hour + $95 for each additional hour (x − 1 additional hours). y = 180 + 95(x − 1) =
180 + 95x − 95 = 95x + 85.
